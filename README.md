## vue-js-toast

> vue toast plugin

## Usage:

```bash
npm i -S vue-js-toast
```

```vue
<Toast content="获取失败" :time.sync="time"></Toast>
```

main.js:
```js
import Toast from 'vue-js-toast'
Vue.use(Toast)
```
